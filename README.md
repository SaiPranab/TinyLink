# TinyLink

TinyLink is a simple, fast, and lightweight URL shortener built with **React** for the frontend and **Express + MySQL** for the backend. Users can create custom short links, track total clicks, and view detailed statistics for each link.

## Features

- Shorten URLs with optional custom codes.
- Dashboard displaying all links in a searchable table.
- Click tracking with total click counts.
- Stats page with detailed link info and timestamps.
- Clean, responsive UI using Tailwind CSS.
- Real-time notifications using React Toastify.

## Tech Stack

- **Frontend:** React, Tailwind CSS, React Toastify, Lucide Icons.
- **Backend:** Express.js, MySQL (or any compatible SQL DB).
- **Client-Side Routing:** Custom hook (`useClientRouter`).
